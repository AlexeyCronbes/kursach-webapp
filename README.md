## Символическая ссылка на хранилище

После переноса проекта на хостинг или другой сервер символическая ссылка `public/storage` не переносится — её нужно создать заново:

```bash
php artisan storage:link
```

Без этой ссылки файлы из `storage/app/public` (загруженные изображения и т.п.) не будут доступны по URL `/storage/...`.
